# Island and mainland bird populations — heterozygosity and range size

Average basepair heterozygosity plotted against the log of range size for endemic island and mainland bird populations.

## Files

- `island_birds_pop_size_pi.csv` — per-population records of range size and genetic diversity

## Variables

| Column | Description |
|--------|-------------|
| `Island_mainland` | Population origin: `I` for island endemic, `M` for mainland |
| `pop_size` | Range size of the population (proxy for population size) |
| `piS` | Average basepair heterozygosity (nucleotide diversity) |
